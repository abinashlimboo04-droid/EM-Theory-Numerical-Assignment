# Electromagnetic Theory – Numerical Assignment

This repository contains numerical implementations of:
- Coulomb's Law
- Gauss's Law
- Biot–Savart Law
- Maxwell's Equations (1D, 2D, 3D)

Language used: C++

## Folder Structure
- emlib/     : User-defined EM libraries
- examples/  : Programs using the libraries
- report/    : LaTeX report

## How to compile
Example:
g++ emlib/coulomb.cpp examples/coulomb_3d.cpp -o coulomb
